============================конструкция логического выбора============================
if - логика любой программы строиться на if(если) else(иначе)
else -- логика этой логической операции строиться если if утверждение для выражения не верно в данном случае выражения,
то мы тогда выводим выражения на else
if (утверждение для выражения) {
    действие;
}
else{
    действие2;
}
