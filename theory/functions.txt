==================================Функции======================================

Функция — это блок кода, 
который выполняет определенную задачу и может быть вызван из других частей программы


Синтаксис функции:

тип_возвращаемого_значения имя_функции(параметры) {
    // тело функции
    return значение;  // если тип не void
}
