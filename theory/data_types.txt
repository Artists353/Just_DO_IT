-----------------ТИПЫ ДАННЫХ-------------------------
есть 3 типа данных и это:
1. -Числовые
2. -Символьные
3. -Логические

Тип=====Размеры в байтах==========Пояснение:
float     4                       Описывает дробные числа 
double    8                       Описывает дробные числа(больше диапозон хранения данных)
int       4                       Описывает целые числа
short     4                       Описывает короткие целые числа
long      4                       Описывает целые числа
char      1                       Описывает символы 
bool      1                       Описывает логические значение(Если поместить 0, то True для компилятора, если 1, то False для компилятора)
